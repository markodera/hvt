# SDKs

The TypeScript SDK has moved to its own public repository:

- SDK repo: [markodera/hvt-sdk](https://github.com/markodera/hvt-sdk)
- Package docs: [docs.hvts.app](https://docs.hvts.app)
- Main app: [hvts.app](https://hvts.app)
- Direct API base URL: [api.hvts.app](https://api.hvts.app)

Use the standalone SDK repo if you want to contribute to `@hvt/sdk`.

If you are not using an SDK, integrate directly against the HTTP API at `https://api.hvts.app`.
